    • Installation

    Copy the archive file ZZZ_courseplay.zip to the folder C:\Users\[Username]\My Documents\MyGames\FarmingSimulator2013\mods. That's all, folks!

    In order to install the developer version, the downloaded zip file needs to be unpacked. The content of the courseplay-master folder must then be packed (archived) into a new zip file by the name of ZZZ_courseplay.zip. After that, proceed as usual (see above).
